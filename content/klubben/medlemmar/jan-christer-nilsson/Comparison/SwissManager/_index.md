## Installation av SwissManager

* [SM Download](https://swiss-manager.at/download.aspx?lan=0)
* Fick en licens av Björn.
* Starta SM
* Other | Install
* Display conditions for use
* Accept conditions for use
* Klistra in licensen, 16 tecken
* OK

## Hämtning av FIDE:s ratingfil

* [FIDE Download](https://ratings.fide.com/download_lists.phtml)
* Rating Lists | Import rating lists
* Import rating list
  * FIDE
  * Ange filen
  * Start import
* OK

## Skapande av turnering

* File | New tournament
* Swiss System
* Tournament name: KM
* Time Control: 90+30
* Federation: SWE
* Rounds: 10
* Time Control: Standard
* Rated FIDE: No
* Rated National: No
* OK

## Hämtning av deltagare

* Input | Enter players
* Input from List
* Rating List: FIDE
* Int. rating from 1400 to 2500
* Date of birth: 1901 to 1960
* Federation: SWE
* Start search
* (nu hittades 704 spelare)

## Urval av deltagare

* Name or Code: Nilsson
* Dubbelklick
* Toggla "check presence of players"
* Name or Code: Flardh
* När alla hittats: OK

## Inspektera deltagarna

* Lists | Starting rank [ctrl-B]

## Lotta nästa rond

* Pairings | Computer Pairings [F6]
* OK
* Start | OK

## Visa Bordslistan

* Lists | Pairings [F10]

## Lägg till rating för båda spelarna

* Print menu
* Define new list
* Gå ner till 10 Rtg.Int. show. 
* Tryck på y
* Gå ner till 28 Rtg.Int. show. 
* Tryck på y
* X End
* Stäng fönster (5)
* F10
* (nu ska rating visas för båda spelarna)

## Mata in resultat

* Input | Enter results [F7]
* **Key** = **Button**
* 0 = Empty
* 1 = 1:0
* 2 = remi
* 3 = 0:1
* 4 = 1F:0F
* 5 = 0F:1F
* 6 = 0F:0F
* 7 = Adjourn
* (övriga via musklick)
* End

## Visa ställning
* Lists | Standings [F5]

## Fortsätt

* Gå tillbaks till Lotta nästa rond
